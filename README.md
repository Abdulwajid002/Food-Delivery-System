# 🍔 Food Delivery System

A modern **MERN Stack Food Delivery System** that allows customers to browse food items, manage their cart, place orders, and make online payments. The project also includes a dedicated **Admin Panel** for managing food items, orders, and users.

---

## 🚀 Features

### 👤 User Panel
- User Registration & Login
- Secure Authentication
- Browse Food Categories
- Search Food Items
- Add to Cart
- Update Cart Quantity
- Place Orders
- Online Payment Integration
- Order History
- Responsive User Interface

### 👨‍💼 Admin Panel
- Admin Authentication
- Dashboard
- Add Food Items
- Update Food Items
- Delete Food Items
- Manage Orders
- Update Order Status
- View Customer Orders

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
  

### Backend
- Node.js
- Express.js
- JWT Authentication
- Multer (Image Upload)

### Database
- MongoDB
- Mongoose

### Payment
- Stripe Payment Gateway

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/Abdulwajid002/Food-Delivery-System.git
```

```bash
cd Food-Delivery-System
```

---

## Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd ../frontend
npm install
```

### Admin Panel

```bash
cd ../admin
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm run server
```

or

```bash
npm start
```

---

### Start Frontend

```bash
cd frontend
npm run dev
```

---

### Start Admin Panel

```bash
cd admin
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

STRIPE_SECRET_KEY=your_stripe_secret_key
```
